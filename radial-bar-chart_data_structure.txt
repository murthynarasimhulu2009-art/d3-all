Explanation of fields:
- category – categorical (nominal) field serving as the identifier for each slice (January‑December). Used for the angular position (startAngle/endAngle) and color encoding.
- value – quantitative field representing the metric to be visualized; mapped through the radius scale to set each bar’s outer radius.
- count – extra quantitative field not currently mapped; could be used for additional visual encodings (e.g., tooltip, size) if needed.

The data array is flat (no nested arrays such as nodes/links). Each object contains the three fields shown above. No hierarchical or linked structures are present.
