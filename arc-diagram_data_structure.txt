Explanation of key parts:
- nodes: collection of points placed along a horizontal axis; each node has an identifier, a group for color, and an index for positioning.
- links: connections between two nodes; each link references source and target node IDs and provides a ready‑made SVG path (arc) that will be drawn as a path mark.
- fields: describes how each property should be interpreted (nominal vs quantitative) and how to access it in the data objects.
